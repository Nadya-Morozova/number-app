# Number App

NumberApp is a simple and fun app where you can find out random facts about your favorite numbers. 
It's simple: enter a number and get a fact or generate a random number with its fact!

## Technologies used

- AndroidX Navigation
- Retrofit
- Coroutines
- Room
- Material Theme
- LiveData
- ViewModel
